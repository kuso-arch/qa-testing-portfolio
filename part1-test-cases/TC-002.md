# TC-002: Негативный сценарий — отклонено банком

**Предусловия:**  
- Пользователь на странице оплаты  
- В корзине товары на сумму > 0  
- Карта валидная (тестовая карта шлюза)

**Шаги:** 
1. Ввести данные карты: номер, срок, CVC  
2. Нажать «Оплатить»  
3. Пройти 3-D Secure (если требуется)  
4. Дождаться редиректа обратно

**Ожидаемый результат:**  
- Статус заказа не меняется  
- Сообщение «Карта отклонена банком»  
- Нет чека на email  
- Корзина остаётся
