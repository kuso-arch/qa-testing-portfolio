# TC-002: Отклонение банком

**ID:** TC-002  
**Название:** Оплата отклонена банком  

**Предусловия:**  
- Корзина заполнена  
- Тестовая карта, отклоняемая банком (из документации шлюза)

**Шаги:**  
1. Ввести данные карты  
2. Нажать «Оплатить»  
3. Дождаться ответа шлюза  

**Ожидаемый результат:**  
- Статус заказа остаётся «Ожидает оплаты»  
- Отображается сообщение «Карта отклонена банком. Свяжитесь с банком»  
- Нет чека на email  
- Товары остаются в корзине
