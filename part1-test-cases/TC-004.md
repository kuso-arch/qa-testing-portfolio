# TC-004: Невалидный номер карты — клиентская валидация

**ID:** TC-004  
**Название:** Клиентская валидация — неверный номер карты  

**Предусловия:**  
- Корзина заполнена  

**Шаги:**  
1. Ввести невалидный номер карты (например, 4111 1111 1111 1112)  
2. Заполнить срок и CVC  
3. Нажать «Оплатить»  

**Ожидаемый результат:**  
- Поле номера подсвечивается красным  
- Сообщение «Неверный номер карты»  
- Запрос к шлюзу не отправляется
