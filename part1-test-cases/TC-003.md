# TC-003: Ошибка платёжного сервиса

**ID:** TC-003  
**Название:** Ошибка платёжного шлюза  

**Предусловия:**  
- Корзина заполнена  
- Тестовая карта, вызывающая ошибку шлюза

**Шаги:** те же  

**Ожидаемый результат:**  
- Сообщение «Ошибка платёжного сервиса. Попробуйте позже»  
- Статус заказа не меняется  
- Нет чека  
- Товары остаются в корзине
