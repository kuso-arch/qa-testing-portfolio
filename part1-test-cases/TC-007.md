# TC-007: Редирект на 3-D Secure и возврат

**ID:** TC-007  
**Название:** Оплата с редиректом на 3-D Secure  

**Предусловия:**  
- Тестовая карта с обязательным 3-D Secure  

**Шаги:**  
1. Ввести данные карты  
2. Нажать «Оплатить»  
3. Пройти 3-D Secure (ввести код)  
4. Вернуться в приложение  

**Ожидаемый результат:**  
- Редирект на страницу 3-D Secure  
- После кода — возврат в приложение с успешной оплатой
