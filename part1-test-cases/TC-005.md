# TC-005: Граничное значение — сумма заказа = 0

**ID:** TC-005  
**Название:** Оплата при пустой корзине  

**Предусловия:**  
- Корзина пуста  

**Шаги:**  
1. Перейти на страницу оплаты  
2. Нажать «Оплатить»  

**Ожидаемый результат:**  
- Кнопка «Оплатить» неактивна  
- Или отображается ошибка «Корзина пуста»  
- Запрос к шлюзу не отправляется
